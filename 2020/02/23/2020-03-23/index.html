Posted on 2020-02-23 Edited on 2020-07-01 In Java

Java程序的类型

Application（应用程序）

  • 独立的程序，需要执行器（虚拟机）来运行

Applet（小应用程序）

  • 由专门的appletViewer运行

  • 或者由Web浏览器调用Java的虚拟机来运行

AppletViewer是jdk工具，位于jdk安装路径/bin中，使用方法与java.exe一样。

下面是一个简单的 Applet 程序 HelloWorldApplet.java

还需要一个HTML网页来调用Applet的代码

Java程序的基本构成

  • package语句

  • import语句

    • 导入其他类的类库

  • 类的定义

    • 一个文件只能由一个public类与文件名相同

  • 类=类头+类体

  • 类成员=属性（attribute）+方法（method）

    • 属性（field，attribute，变量，字段） 方法（method，函数）

  • 方法=方法头+方法体
