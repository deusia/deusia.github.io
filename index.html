关于DDR和LPDDR

Posted on 2021-12-12 In 硬件

关于DDR和LPDDR

首先这些名词都是和内存有关系的。前者是常常在PC装机的时候出现的。还记得我第一次给我的联想电脑加内存的时候，我就查了我的电脑是DDR2还是DDR3？我记得应该是DDR3的。然后我就买了DDR3的内存条，在买电脑的联想店里面。但是那个时候的那家店，好像就成了家挂着羊头卖狗肉的店，估计是倒了吧。

那关于LPDDR这个词儿，我是在使用手机的时候知道的。

其实用DDR和LPDDR来描述都是不准的。有这么一个词儿——DDR SDRAM。

DDR SDRAM全称为Double Data Rate Synchronous Dynamic Random Access Memory

中文的意思是双倍数据率同步动态随机存取存储器

传统的SDRAM在一个时钟周期内只传输一次资料，它是在时钟上升期进行资料传输；而DDR SDRAM则是一个时钟周期内可传输两次资料，也就是在时钟的上升期和下降期各传输一次资料。

简化的称DDR SDRAM为DDR。

那关于LPDDR，是Lower Power Dynamic Random Access Memory的意思。它是一种DDR，也是移动DDR SDRAM。这里特别强调Lower Power的目的是这类的内存是用于移动、编写设备的。比如手机、iPad、和笔记本电脑这种设备。这里的内存以更低的电压和电流驱动内存，以达到更加省电的效果，来为设备提供更长的设备续航。

关于时序

时序在电路中有着至关重要的作用。做个比喻就是乐谱中的节拍。再一个比喻就是，大家都多少听过，一二三木头人这个游戏。

关于时钟频率。如果我们每一秒中眨一次眼睛，那时钟频率为10Hz，则每隔0.1秒就眨眼一次，1秒钟就眨眼十次。目前我们的CPU的时钟频率是以多少GHz计的。如果是2.4GHz，那CPU会眨眼的话，1秒钟眨眼
$$
2.4*10^6次
$$
笔者的M1处理器，Efficiency Cores的时钟频率为2.0GHz，Performance Cores的时钟频率为3.2GHz。

只要CPU电路时序能够做到更高，那CPU能做的也越多，相应的CPU的应能也越强。时序的提高对物理电路的影响，会让电路发出热量，过热导致电路被烧坏。

在CPU的设计和性能方面，我们提高CPU的时钟频率，同时使用更小的制程（更小的晶体管）来提升CPU的性能。让CPU在性能和发热之间达到一种平衡。更小的制程，晶体管更小，能在一定的单位面积下提供更大的堆积规模以及更小的电压和电流，从而再保证更强的性能的同时维持更低的功耗，让热量更少。

为什么提热量和功耗？

未来是移动设备的天下，越来越多的设备会更加着重的考虑移动性。而发续航、发热、性能都是在设备移动上必须要解决的问题。性能强而发热少，功耗低而续航长，这是未来的便携式移动设备要做到的。

An algorithm is a step-by-step procedure for solving a problem

Posted on 2021-12-10

An algorithm is a step-by-step procedure for solving a problem.

“Pattern of thought” which governs scientific apprehension during a certain period of time.

General approaches to the construction of efficient solutions to problems

暴力破解法（Brute Force Paradigm）

暴力破解法简单直接，根据问题声明的定义，找到所有可变化的因子（Divisor），穷举所有可能解决问题的方法，逐个尝试。
所以，根据暴力破解法的定义，理论上讲任何问题都可以使用暴力破解法来解决，只是在实际应用中算法对时间和空间的需求则无法满足。
将暴力破解法应用于数据查找，由于查找比较次数与给定目标的规模直接相关，所以也称为线性查找（Linear Search）。
线性查找有很多典型应用：

  • 选择排序（Selection Sort）
  • 冒泡排序（Bubble Sort）
  • 顺序查找（Sequential Search）
  • 暴力字符串匹配（Naive String Match）
